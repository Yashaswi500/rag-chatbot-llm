# 🤖 RAG Chatbot (LLM-Based)

End-to-end Retrieval-Augmented Generation (RAG) chatbot for document question answering using LangChain, FAISS, and Hugging Face models.

---

## 🚀 Features

* 📄 Upload PDF and ask questions
* 🔍 Semantic search using FAISS
* 🧠 LLM-based answer generation
* ⚡ Real-time interaction using Streamlit
* 🎯 Reduced hallucinations with prompt tuning

---

## 🛠 Tech Stack

* Python
* LangChain
* FAISS
* Hugging Face Transformers
* Streamlit

---

## ⚙️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 📌 Use Case

This project demonstrates how LLMs can be combined with external knowledge (RAG) to provide accurate, context-aware answers from custom documents.
